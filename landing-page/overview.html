Overview

A terminal-first coding agent: read, write, edit, and run commands in your project — powered by Sarvam AI out of the box. Built on the open-source Pi toolkit.

What is Supratim?

🎯

Dogfood-First

Daily use in real repos is the only benchmark. Every phase ships something you'll actually use.

🔌

Model-Agnostic

Sarvam showcased by default. Swap to any provider or local model — Pi is already BYOK.

🧩

Depend, Don't Fork

Built on Pi packages. Stays aligned with upstream — we extend, not duplicate.

What's Ready Today

Phases 0 & 1 are fully shipped and published.

🔑

Secure Key Onboarding

TUI wizard on first run. Keys stored in Windows Credential Manager / macOS Keychain via keytar — never written to disk.

📊

Live Usage HUD

Real-time token counts, running cost in ₹, context window %, current model — always visible in the footer.

🛠️

Core Tools

read, write, edit, bash — all scoped to your project directory via Pi.

⚡

Sarvam Optimised

Compat extension flattens Pi content blocks; handles Sarvam tier max_tokens limits automatically.
